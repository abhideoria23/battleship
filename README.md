# battleship

Thi is is a double player BattleField Game.


Not: Ships can be placed Horizontally and vertically
Board Size is 10*10
Boards Marking :
0 : No ship
1 : Target Hit
2 : Target Missed
C : Carrier
B : BattleShip
c : Cruiser
S : Submarine
D : Destroyer

Starting point of the game is play class
